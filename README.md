
# ☕ COFFIENS - 지역 카페 기반 구독·주문 통합 플랫폼

> COFFIENS는 지역 카페를 위한 커피 구독·주문 통합 플랫폼입니다. <br>
소비자에게는 월정액 커피 구독과 간편한 주문 경험을 제공하고,
점주에게는 예측 가능한 매출과 단골 확보 기능을 제공합니다. <br>
소규모 카페와 소비자를 연결해 지역 상권을 활성화하는 것을 목표로 합니다.
